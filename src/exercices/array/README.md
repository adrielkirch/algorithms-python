## How to Choose the Right Technique

- **Use HashMaps** when you need fast lookups, counting frequencies, or detecting duplicates
- **Use Two Pointers** when working with sorted arrays or need to find pairs/triplets
- **Use Greedy** when local optimal choices lead to global optimum
- **Use Sliding Window** when dealing with subarrays or substrings of fixed/variable size
- **Use Binary Search** when data is sorted or you can define a search condition
- **Use Simulation** when you need to model real-world processes step by step
