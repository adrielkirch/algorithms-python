## üöÄ Getting Started

### Prerequisites

```bash
Python 3.x
Jupyter Notebook (for .ipynb files)
```

### Installation

1. Clone the repository:
```bash
git clone https://github.com/adrielkirch/algorithms-python.git
cd algorithms-python
```

2. Install Jupyter Notebook (if not already installed):
```bash
pip install jupyter
```

3. Launch Jupyter Notebook:
```bash
jupyter notebook
```

### Running the Code

Each algorithm and problem is contained in its own Jupyter Notebook (`.ipynb`) file with:
- **Problem Statement**: Clear description of the challenge
- **Solution**: Implementation with comments
- **Test Cases**: Multiple test cases to verify correctness
- **Explanation**: Strategy and approach documentation

Simply open any `.ipynb` file and run the cells sequentially.

## üìñ How to Use This Repository

### For Interview Preparation

## üéì Learning Path

**Beginner**: Start with:
- Basic data structures (array, linked list, stack, queue)
- Simple sorting algorithms (bubble, insertion, selection)
- Basic search algorithms (linear, binary)

**Intermediate**: Move to:
- Advanced sorting (merge sort, quick sort)
- Hash tables and their applications
- Tree traversal and basic tree problems
- Two pointers and sliding window techniques

**Advanced**: Master:
- Dynamic programming problems
- Graph algorithms (BFS, DFS, Dijkstra, A*)
- Complex data structure combinations
- Optimization techniques
